# Overview

OrbisDB is a fully open-source relational database built on top of Ceramic, a decentralized event store. It inherits data ownership, composability and decentralized properties of Ceramic, while providing a highly scalable way of querying data via SQL, GraphQL and a built-in Dashboard.
